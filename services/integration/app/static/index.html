Enterprise Integration Patterns

1 Message Channel
Queue RabbitMQ dedikasi per domain: patient.registration, prescription.created, billing.insurance, billing.cash
2 Publish-Subscribe
Event registrasi pasien dipublish ke fanout exchange, diterima oleh multiple consumer (EMR + Billing)

4 Message Translator
